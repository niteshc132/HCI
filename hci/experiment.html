Experiment details

We want to compare how different participants perform using the drawing tablet compared with hand drawn. We also want to see if their performance compares whether they commonly use a drawing tablet.


Hypothesis


Participants that have no prior experience with a drawing tablet would prefer pen and paper as that’s what they are familiar and comfortable with.

We assume that the drawing tablet would have a learning curve to it, therefore people that are familiar and commonly use drawing tablets would prefer the tablet over hand drawing.


Method

We are going to set up a comparison test where a volunteer participant will use a drawing tablet and pen and paper to draw multiple shapes and then compare the outcome of their drawing to “perfect” examples to rate them based on accuracy. We will also ask each participant which they prefer. The drawing tablet we will be using will be the Surface Pro 6 with Surface Pen.

On the 10th of October, during the time period of 2:00pm - 4:00pm, we conducted our experiment.


Variables

Independent Variable:

  • Surface Pro 6 with Surface Pen using Photoshop CC
  • Ball-point pen on paper/li>

Dependent Variables:

  • Mouse Accuracy Score - This is a score measured by mouseaccuray.com.
  • Comfortable Rating - A rating determined by a survey measured between 1 and 5.

Controlled Variables:

  • Drawing Tablet - All tests will be completed of the same Surface Pro 6 using the included stylus. All tests using pen and paper will be done with the same pen.
  • Software - All tests will be completed with the participants using Photoshop CC to draw.
  • Brightless and Slope - All tests will be completed with the tablet in the same position with the same brightness levels.

Random Variable:

  • Temperature - The temperature of the room can not be controlled as we have no control over the air conditioning.

In order to conduct this test we will be using a mouse accuracy test at www.mouseaccuracy.com. This website contains a mini-game that is supposed to help train mouse accuracy. We will be using the default difficulty levels.

All data for each participant is being compared to their tests on the other laptops. The order of laptops being used will be determined randomly.
